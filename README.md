# FlappyBirds

Pet project: 2D arcade-игра, разработанная на Unity с использованием C#.

## О проекте

Этот проект представляет собой 2D-игру, вдохновлённую механикой Flappy Bird.  
Он был создан как pet project для практики разработки игр на Unity, работы с игровой логикой, пользовательским интерфейсом и структурой проекта.

## Реализовано

- управление персонажем
- движение с использованием физики Unity
- обработка столкновений
- логика завершения игры
- система сбора монет
- отображение количества монет в UI
- рандомизация препятствий
- экран Game Over
- перезапуск уровня
- улучшение стартового меню
- работа со сценами

## Стек технологий

- C#
- Unity
- Unity UI
- Git

## Структура проекта

- Assets/Scenes — сцены проекта
- Assets/Scripts — скрипты с игровой логикой
- Assets/Prefabs — префабы игровых объектов
- Assets/Animations — анимации
- Assets/Sprites — визуальные ресурсы

## Как запустить проект

1. Склонировать репозиторий
2. Открыть проект в Unity
3. Загрузить основную сцену
4. Запустить проект через Play Mode
